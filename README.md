# Number Printer

Program Python yang mencetak angka dari 1-100 dalam urutan terbalik dengan aturan khusus.

## Deskripsi

Program ini mencetak angka dari 100 hingga 1 dengan ketentuan:
- Tidak mencetak bilangan prima
- Angka yang habis dibagi 3 diganti dengan "Foo"
- Angka yang habis dibagi 5 diganti dengan "Bar"
- Angka yang habis dibagi 3 dan 5 diganti dengan "FooBar"
- Output dicetak secara horizontal dengan pemisah koma

## Contoh Output

```
Bar, Foo, 98, Foo, Bar, 94, Foo, 92, 91, FooBar, 88, Foo, 86, Bar, Foo, 82, Foo, Bar, Foo, 77, 76, FooBar, 74, Foo, Bar, Foo, 68, Foo, Bar, 64, Foo, 62, FooBar, 58, Foo, 56, Bar, Foo, 52, Foo, Bar, 49, Foo, 46, FooBar, 44, Foo, Bar, Foo, 38, Foo, Bar, 34, Foo, 32, FooBar, 28, Foo, 26, Bar, Foo, 22, Foo, Bar, Foo, 16, FooBar, 14, Foo, Bar, Foo, 8, Foo, 4, 1
```

## Cara Menjalankan

1. Pastikan Python sudah terinstall
2. Jalankan program dengan perintah:
   ```bash
   python number_printer.py
   ```

## Requirements

- Python 3.x
